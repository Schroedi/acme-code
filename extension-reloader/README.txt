extension-reloader


Gnome Shell Extension Reloader

This extension ceates a menu of the installed extensions and executes
gnome-shell-extension-tool for the selected extension from the menu
to reload the extension.  This extension functions for both Xorg and
Wayland Gnome Shell sessions.  This extension is intended for use by
extension authors.  General use of this extension is not anticipated.
